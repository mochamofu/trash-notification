# ゴミスタンプ

地域ごとのゴミ出し周期を、スタンプを置く感覚で設定する生活リマインダーのプロトタイプです。

前日夜と当日朝にローカル通知で知らせることで、「明日プラの日だった」「今日可燃ごみの日だった」を忘れにくくするコンセプトです。

## Main Flow

1. 初回セットアップ
2. ゴミ種別スタンプを曜日へ配置
3. 毎週、隔週、第n曜日などの周期を設定
4. 前日22:00、当日7:00の通知時刻を設定
5. 今日と明日のゴミ出し予定を確認
6. ゴミ出し完了後に広告を表示
7. 360円の買い切りで広告を解除

## Interactive Simulator

`gomi-stamp-interactive.html`では以下をクリックで試せる。

- 初回起動時のセットアップ
- ゴミ種別の選択
- 曜日ごとのゴミスタンプ配置
- 週間サイクルの保存
- 今日画面への反映
- ゴミ出し完了チェック
- 完了後広告
- 360円の広告解除
- ライト/ダークモード切り替え

## Design Direction

日常的に開く生活アプリとして、硬い管理画面ではなく親しみやすい見た目を目指す。

- 明るい緑を中心に、白いカードと淡い背景で軽さを出す
- ダークモードでは黒一色ではなく、深い緑系で落ち着いた印象にする
- 週間サイクルの入口は今日画面上部に置く
- 広告解除や内部方針は必要以上に設定画面へ出さない
- 操作ボタンは短く、日常の確認を邪魔しない

## Monetization

無料版では、操作を邪魔しにくい位置とタイミングに広告を出す。

- 下部バナー広告
  - 今日画面、予定画面、設定画面の下部に常時表示
  - 下タブの上に固定し、主要操作には重ねない
- 完了後広告
  - ゴミ出し確認、設定保存など、用事が終わった後に表示
  - アプリ起動直後や通知タップ直後には出さない
- 広告解除
  - 買い切り360円
  - 下部バナー広告と完了後広告を非表示にする

起動時広告は、急いでゴミ出し確認をしたい体験と相性が悪いため採用しない。

## Cost Policy

初期版は外部APIやサーバーを使わず、端末内保存とiOSローカル通知で成立させる方針です。

- 外部API費用: 0円想定
- サーバー費用: 0円想定
- 通知: iOSローカル通知
- 保存: 端末内ストレージ
